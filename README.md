# API Learning Project (Node.js)

A simple JavaScript client that demonstrates how to perform CRUD (Create, Read, Update, Delete) operations against a REST API.

This project was built to learn **API integration**, **HTTP Methods**, and **Asynchronous JavaScript** using Node.js.

## üöÄ Technologies Used
* **JavaScript** (ES6+)
* **Node.js** (Runtime environment)
* **Fetch API** (Native Node.js implementation)
* **JSONPlaceholder** (Fake Online REST API for testing)

## üìã Features
The script `index.js` performs the following automated tasks:
1.  **GET**: Fetches a post from the server.
2.  **POST**: Simulates creating a new post with JSON data.
3.  **PUT**: Updates an existing post.
4.  **DELETE**: Removes a post from the server.
